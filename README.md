🤖 Bot de Discord com IA

Bem-vindo ao meu projeto! 🚀

Este é um bot de Discord desenvolvido em Python que reúne diversão, inteligência artificial e utilidades em um só lugar. A ideia é criar um bot cada vez mais completo, adicionando novos recursos e tornando a experiência dos usuários mais interativa. 🎮✨

🌟 Funcionalidades atuais

Atualmente, o bot possui os seguintes comandos:

✅ Status do Bot
Verifica se o bot está online e funcionando corretamente. 🟢

📝 Simulado Interativo
Cria um simulado com 10 questões, calcula a pontuação e transforma o aprendizado em uma experiência parecida com um jogo. 📚🏆

🧠 Análise de Imagens com Inteligência Artificial
Utiliza IA para analisar imagens e identificar informações presentes nelas. 🤖📷

🎲 Número Aleatório
Gera um número aleatório entre 1 e 100 para brincadeiras, desafios ou sorteios. 🔢

😀 Emoji do Dia
Envia um emoji que representa a pessoa naquele dia, trazendo uma interação mais divertida. ✨
